# Dark Developer Portfolio Template

## Style
Moody, dramatic dark theme with neon green accents. Terminal/matrix aesthetic for developers who want to stand out.

- **Font**: Syne (headings) + Space Mono (mono/UI)
- **Color Palette**: Near-black (#050508) background, Neon Green (#00ff87) + Cyan accents
- **Aesthetic**: Terminal, hacker, editorial dark

## Features
- Scanline overlay texture for authentic terminal feel
- Grid background pattern in hero
- Live JSON terminal widget in hero section
- Neon glow effects on interactive elements
- Monospace UI labels throughout
- Tab-based experience/education switcher
- Neon-bordered project cards with hover glow
- Dark form with monospace labels

## Setup
1. Replace all `{{PLACEHOLDER}}` values with your actual data
2. Add your profile photo at `assets/img/profile-pic1.png`
3. Update project images in `assets/img/`
4. Replace resume PDF at `assets/My Resume 1.pdf`
5. Use `portfolio-integration.py` to auto-fill from your resume

## Placeholders Reference
Same placeholder system as the tech template — fully compatible with `portfolio-integration.py`.
