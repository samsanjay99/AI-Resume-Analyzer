# Professional Portfolio Template

## Style
Clean, elegant, light corporate theme with gold accents. Built for professionals who want a polished, trustworthy first impression.

- **Font**: Playfair Display (headings) + DM Sans (body)
- **Color Palette**: White/Cream background, Navy primary, Gold accent (#c9a84c)
- **Aesthetic**: Editorial, refined, corporate

## Features
- Diagonal hero split layout with portrait frame
- Section eyebrow labels with gold rule
- Professional timeline with dot indicators (side-by-side columns)
- Pill-style skill badges
- Project cards with overlay actions
- Clean contact form with labeled fields
- Animated scroll reveals

## Setup
1. Replace all `{{PLACEHOLDER}}` values with your actual data
2. Add your profile photo at `assets/img/profile-pic1.png`
3. Update project images in `assets/img/`
4. Replace resume PDF at `assets/My Resume 1.pdf`
5. Use `portfolio-integration.py` to auto-fill from your resume
