## Interaction Flow

### Login Flow
```
1. User sees animated background
2. Glass container slides up
3. Logo pulses to grab attention
4. User hovers over input → highlights
5. User types → green focus border
6. User hovers button → lifts up
7. User clicks → button presses
8. Success → balloons animation
```

### Signup Flow
```
1. User clicks "Create Account"
2. Page transitions to signup
3. Different logo (✨ instead of 🚀)
4. Same beautiful effects
5. More input fields
6. Same smooth interactions
7. Success → auto-login + balloons
```
