## Placeholder Compatibility

✅ **All templates use identical placeholders**, ensuring seamless switching between templates:

### Personal Information
- `{{FULL_NAME}}` - Complete name
- `{{FIRST_NAME}}` - First name only
- `{{EMAIL}}` - Email address
- `{{PHONE}}` - Phone number
- `{{LOCATION}}` - Location/city

### Professional Details
- `{{JOB_TITLE}}` - Current job title
- `{{CURRENT_POSITION}}` - Current position
- `{{PROFESSIONAL_SUMMARY}}` - Professional summary
- `{{ABOUT_DESCRIPTION}}` - About section text

### Statistics
- `{{YEARS_EXPERIENCE}}` - Years of experience
- `{{PROJECT_COUNT}}` - Number of projects
- `{{SKILL_COUNT}}` - Number of skills

### Social Links
- `{{LINKEDIN_URL}}` - LinkedIn profile
- `{{GITHUB_URL}}` - GitHub profile
- `{{TWITTER_URL}}` - Twitter profile
- `{{RESUME_DOWNLOAD_LINK}}` - Resume download link

### Content Sections
- `{{PROGRAMMING_LANGUAGES}}` - Programming languages HTML
- `{{FRAMEWORKS}}` - Frameworks HTML
- `{{DATABASES_TOOLS}}` - Databases & tools HTML
- `{{CLOUD_DEVOPS}}` - Cloud & DevOps HTML
- `{{WORK_EXPERIENCE}}` - Work experience HTML
- `{{EDUCATION_HISTORY}}` - Education history HTML
- `{{PROJECTS_LIST}}` - Projects list HTML
- `{{CONTACT_MESSAGE}}` - Contact section message

### Typewriter Effect
- `{{SECONDARY_TITLE_1}}` - Alternative title 1
- `{{SECONDARY_TITLE_2}}` - Alternative title 2
- `{{SECONDARY_TITLE_3}}` - Alternative title 3

## AI Data Extraction

All templates work with the same AI-powered data extraction system:

1. **Resume Upload** → Text extraction
2. **AI Analysis** → Gemini API processes resume
3. **Data Extraction** → Structured JSON output
4. **Placeholder Replacement** → Template population
5. **Preview Generation** → Live preview
6. **Download/Deploy** → ZIP file or Netlify hosting

The AI system:
- Extracts actual data from resume (no fallbacks)
- Calculates statistics (years, projects, skills)
- Formats content for HTML display
- Generates professional summaries
- Infers missing data intelligently
