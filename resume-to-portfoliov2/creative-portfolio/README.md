# Creative Portfolio Template

## Style
Bold, maximalist editorial design with warm coral/peach/orange gradient mesh. For designers, creatives, and developers who want to stand out and be remembered.

- **Font**: Fraunces (expressive serif headings) + Plus Jakarta Sans (body)
- **Color Palette**: Warm white background, Coral (#ff6b35) + Peach + Rose accents
- **Aesthetic**: Maximalist editorial, organic blobs, animated, magazine-style

## Signature Features
- Custom animated cursor (dot + trailing ring)
- Floating animated gradient blobs in hero
- Grain texture overlay for tactile feel
- Floating pill navbar (not full-width)
- Oversized outlined + italic typography combination
- Animated loading counter (000 → 100)
- Skill chips with hover tilt animations
- Card-based timeline for experience/education
- Dark contact section with frosted glass form
- Smooth scroll reveal on all sections

## Setup
1. Replace all `{{PLACEHOLDER}}` values with your actual data
2. Add your profile photo at `assets/img/profile-pic1.png`
3. Update project images in `assets/img/`
4. Replace resume PDF at `assets/My Resume 1.pdf`
5. Use `portfolio-integration.py` to auto-fill from your resume

## Placeholders Reference
Fully compatible with the same `{{PLACEHOLDER}}` system used across all templates.
Works directly with `portfolio-integration.py` and `template-generator.js`.
