## Getting Started

### Creating an Account

1. Click **"Sign Up"** on the login page
2. Enter your email and password
3. Click **"Create Account"**
4. You'll be automatically logged in

### Logging In

1. Enter your email and password
2. Click **"Login"**
3. Access all features from the sidebar

## Resume Analysis

### AI-Powered Analysis

1. Navigate to **"AI Resume Analyzer"**
2. Upload your resume (PDF, DOCX, or TXT)
3. Select analysis mode:
   - **Smart Analysis** - Fast (1-2 seconds)
   - **Deep Analysis** - Comprehensive (5-15 seconds)
4. Click **"Analyze Resume"**

### Understanding Your Score

- **Resume Score**: Overall quality (0-100)
- **ATS Score**: Applicant Tracking System compatibility
- **Keyword Match**: Relevance to target role
- **Format Score**: Structure and organization

### Recommendations

The analysis provides:
- Missing skills to add
- Formatting improvements
- Content suggestions
- Industry-specific tips

## Portfolio Generator

### Creating Your Portfolio

1. Go to **"Portfolio Generator"**
2. Upload your resume
3. Select a template style:
   - Tech Style (default)
   - Dark Developer
   - Creative
   - Professional
4. Click **"Generate Portfolio"**

### Preview & Download

- **Preview**: View your portfolio in the browser
- **Download**: Get a ZIP file with all files
- **Host Online**: Deploy to Netlify instantly

### Customization

The portfolio includes:
- Professional summary
- Skills showcase
- Work experience
- Projects
- Education
- Contact information

## Troubleshooting

### Upload Issues

**Problem**: File won't upload
**Solution**: 
- Check file size (max 10MB)
- Ensure correct format (PDF, DOCX, TXT)
- Try a different browser

### Analysis Errors

**Problem**: Analysis fails
**Solution**:
- Check internet connection
- Verify file is readable
- Try Smart Analysis instead of Deep

### Portfolio Generation

**Problem**: Portfolio looks incorrect
**Solution**:
- Ensure resume has all sections
- Try a different template
- Re-upload resume
